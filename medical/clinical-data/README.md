# Clinical Data Extractor

Herramienta para extracción masiva de datos del sistema Integrador del hospital.

## Configuración

1. Conectar VPN al hospital
2. Configurar credenciales de acceso
3. Ejecutar scripts de extracción

## Estructura

- `config/` - Archivos de configuración
- `src/` - Scripts de extracción
- `data/` - Datos extraídos (no se sube a git)
- `logs/` - Logs de ejecución
